    POST
    Create person

    To create a person you have to pass a first name and a last name.

    POST plugin/wp_elasticio_backend/json?method=insert_contact

    Optional parameters

    Name /type

    Description /example

    Constraints

    firstname
    string

    First name of the person

    "John"

      name
      string

      Name of the person

      "Doe"

        cURL

        curl -X POST "https://oihwice.wice-net.de/plugin/wp_elasticio_backend/json?method=insert_contact" \
     -H "Content-Type: application/json" \
     -H "X-API-KEY: [YOUR_API_KEY]" \
     -H "cookie: [YOUR_COOKIE]" \
     --data '{"firstname":"John","name":"Doe"}'

        Response

        {
  "rowid": "410849",
  "firstname": "John",
  "name": "Doe"
}

            POST
            Delete person

            To delete a person you have to pass person's rowid, which you want to delete. We do NOT really delete the person from our database, we just mark the person as 'deactivated'.

            POST plugin/wp_elasticio_backend/json?method=delete_person

            Optional parameters

            Name /type

            Description /example

            Constraints

            rowid
            string

            User's rowid

            "410849"

              cURL

              curl -X POST "https://oihwice.wice-net.de/plugin/wp_elasticio_backend/json?method=delete_person" \
     -H "Content-Type: application/json" \
     -H "X-API-KEY: [YOUR_API_KEY]" \
     -H "cookie: [YOUR_COOKIE]" \
     --data '{"rowid":"410849"}'

              Response

              {
  "msg": "Person was deleted",
  "rowid": "410849",
  "firstname": "John",
  "name": "Doe"
}
